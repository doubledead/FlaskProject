# FlaskProject
[Flask][fl] app using modular blueprints.

Features:

- [Flask-Security][fs]
- [Jinja2][jj] templating

This webapp requires these programs: 

- Python 2.7.*
- pip
- virtualenv
- bower

[fl]: http://flask.pocoo.org/
[fs]: https://github.com/mattupstate/flask-security
[jj]: http://jinja.pocoo.org/
